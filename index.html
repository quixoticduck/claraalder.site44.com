Codepen

I created this CodePen Pen of an interactive spinning windmill logo as an option for the navigational bar of the Bookkeeping website above. In the end we went with a static logo—based on the Round Hill windmill that was dismantled just over a century ago in 1913—but I liked how it turned out. Hover over the sails of the windmill to see them spin and view the HTML and CSS code by clicking the tabs on the left.
